# 🧠 Data Science & AI Internship Programme

**Duration:** 12 Weeks | **Format:** Online + Self-Paced | **Tools:** Google Colab, Python  
**Level:** Beginner to Intermediate | **Mode:** Group-based Internship Simulation

---

## What Is This Programme?

This is not just a course. It is a **12-week internship simulation** designed to take you from zero data knowledge to a working data scientist who can contribute to real projects.

You will **learn, practice, build, and simulate** real internship work — week by week — using Python on Google Colab. No complex setup. No expensive software. Just a browser and the will to learn.

---

## Who Is This For?

- Engineering students with basic programming knowledge
- Anyone who wants to break into data science or AI
- MERN Stack students who want to add data skills to their portfolio
- People who want to understand how AI actually works under the hood

## 🏗️ Capstone Projects (Group Work)

Each project is designed for a group of 3–4 students. These are MVP-level, production-minded builds.

### Project 1 — Sales Intelligence Dashboard
**Focus:** EDA + Visualization + Basic Forecasting  
A data dashboard that ingests sales CSVs, runs analysis, and surfaces insights + a simple revenue trend predictor.

### Project 2 — Sentiment Analyzer Tool
**Focus:** NLP + ML Classification  
Build a tool that takes customer reviews or social media text and classifies sentiment (positive / negative / neutral) with confidence scores.

### Project 3 — Student Performance Predictor
**Focus:** ML Pipeline + Data Storytelling  
Predict whether a student will pass or need intervention, based on attendance, grades, and activity data. Present findings like a real data team.

## How Each Week Works

Every week follows the same internship-style rhythm:

```
📖 Learn     → Concepts + reading (30 mins)
💻 Practice  → Follow-along Colab notebook (60 mins)
🛠️ Build     → Mini project or exercise (60–90 mins)
🔁 Review    → Reflect, share, get feedback
```

---

## AI Policy

- Weeks 1–4: **AI assistance limited.** Build fundamentals yourself first.
- Weeks 5–12: **AI is your collaborator.** Use it to accelerate — not to skip understanding.
- Golden rule: If you cannot explain what the code does, you do not own it.
