# Week 12 — Capstone Deployment

## Objective

Deploy full production system.

Students will:

• deploy infrastructure  
• implement CI/CD  
• enable monitoring  

---

# Capstone Requirements

Project must include:

• Docker containers  
• AWS deployment  
• CI/CD pipeline  
• monitoring system  

---

# Final Deliverables

Students must submit:

1. Live deployed application
2. GitHub repository
3. Infrastructure documentation
4. Deployment pipeline
5. Architecture diagram

---

# Evaluation Criteria

| Criteria | Weight |
|------|------|
| Architecture | 25% |
| Deployment | 25% |
| CI/CD | 20% |
| Monitoring | 15% |
| Documentation | 15% |

# Final Outcome

Students graduate with **production DevOps portfolio project**.
