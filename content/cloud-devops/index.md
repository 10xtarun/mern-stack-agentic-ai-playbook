# Cloud & DevOps Engineering Internship (12 Weeks)

## Program Overview

This program is a **12-week Cloud and DevOps internship** designed to transform developers into **deployment-ready engineers** capable of building, shipping, and operating production systems.

The course follows a **learning-by-building approach**, where interns will:

• Learn DevOps principles  
• Build CI/CD pipelines  
• Containerize applications  
• Deploy scalable systems on AWS  
• Monitor and maintain production infrastructure  

By the end of the internship students will deploy a **production-ready cloud system**.

---

# Program Structure

The course is divided into **two phases**.

## Phase 1 — Foundations (Weeks 1–6)

Focus: **Core DevOps Skills**

Students will learn:

- Git workflows
- CI/CD automation
- Docker containerization
- Multi-container applications
- Cloud infrastructure basics

Outcome:

Students can **containerize and deploy applications to cloud infrastructure.**

---

## Phase 2 — Application & Production Systems (Weeks 7–12)

Focus: **Production DevOps Engineering**

Students will learn:

- Production Docker practices
- AWS container orchestration
- CI/CD pipelines for cloud
- Monitoring and logging
- Infrastructure architecture

Outcome:

Students can **design, deploy and maintain scalable production systems.**

---

# Learning Methodology

Each week includes:

1. Concept Learning
2. Hands-on Labs
3. Practice Exercises
4. Mini Projects
5. Documentation
6. Weekly Review

Students maintain a **daily engineering diary**.

By the end of the program students will have:
