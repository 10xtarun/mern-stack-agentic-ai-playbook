AI Policy

When you can use AI — and when you can't

🚫
Weeks 1–4: No AI
Build the fundamentals yourself. The muscle memory of debugging, writing loops, and cleaning data cannot be shortcut. You will struggle. That is the point.
✅
Weeks 5–12: AI as Collaborator
Use Gemini, ChatGPT, Claude. Accelerate — don't skip. The golden rule: if you cannot explain what the code does, you do not own it.

Group Projects

3 Capstone Projects

Teams of 3–4. MVP scope. Real datasets. You present findings like a working data team — not like students submitting homework.

📊
Project 1
Sales Intelligence Dashboard

Ingest 3 years of retail sales data. Surface insights: growth trends, regional gaps, discount impact, profit margins. Build a 3-month revenue forecast. Deliver an analysis report that a business team can act on.

EDA Visualization Time Series Linear Regression Plotly
💬
Project 2
Sentiment Analyzer Tool

Classify customer reviews as Positive, Negative, or Neutral with confidence scores. Handle imbalanced classes. Demo the model on live input. Add AI-generated explanations for each prediction.

NLP TF-IDF Logistic Regression LinearSVC Gemini API
🎓
Project 3
Student Performance Predictor

Predict whether a student is at risk before exams, using attendance, grades, and behaviour data. Generate AI-powered counsellor reports. Explain findings in plain English for a non-technical school administrator.
